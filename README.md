
# YOU-I
# 서비스 소개 / 서비스 설명
- 지역별, 취미별로 그룹을 만들어 모임을 가질수 있는 기능제공
- 그룹별 커뮤니티 사이트 제공
- 사용자의 취미에 평가가 좋은 그룹순으로 볼수 있음.
